Description

Why I Made This Game

  • To stay committed to game development and continue learning during a one-year break, including recovery from wrist surgery.
  • Leveraged ChatGPT to assist with development and improve rapid prototyping skills required of game designers.
  • Challenged myself to create a hyper-casual game that emphasizes individual skill in short play sessions.

Development with ChatGPT

  • Developed core gameplay features and overall UI functionalities.
  • Created an obstacle placement tool using Unity Editor scripting for efficient level design.
  • Optimized performance through object pooling implementation.
  • Implemented an achievement system and a character purchase feature using in-game currency to support character collection.

Business Model

  • In-game advertising (Ad-based monetization).
