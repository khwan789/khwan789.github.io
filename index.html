의도

  • 플레이어 선택에 따라 전투의 향방이 달라지도록 하여 기존에 한마리의 보스와의 전투에서 벗어난 새로운 경험 제공.
  • 속성 간의 유불리한 차이점이 적은 방향으로 던전 제작.

방향

  • 3분의 제한 시간 내에 두 마리의 보스 : 슈텐 도지, 이바라키 도치를 처치.
  • 슈텐 도지, 이바라키 도지, 그리고 5개의 텐트가 맵에 생성된다.
  • 슈텐 도지 우선 처치 시, 보다 약한 이바라키 도지와의 전투를 진행할 수 있으나 빠르게 처치하지 못한다면 슈텐 도지를 회복시켜 다시 전투에 참여하게 할 수 있다.
  • 이바라키 도지 우선 처치 시, 보다 단순한 패턴을 보유한 슈텐 도지와 전투를 진행할 수 있으나, 슈텐 도지가 페이즈 2에 돌입하게 되면서 스킬이 강화되어 피하지 못하였을경우의 리스크가 증가한다.
  • 전투 도중 텐트가 파괴되면 작은 도깨비들이 소환되어 전투에 참여한다. 몬스터들을 보다 효율적으로 처치하기 위해 각기 다른 속성의 표창을 보유한 플레이어들이 협력하게 하여 멀티플레이 요소를 강화.

슈텐 도지

  • 플레이어를 쫓아가며 근접 전투 진행
  • 이바라키 도지가 살아있는 한 죽지 않으며 체력이 ‘0’이 되면 기절 상태 돌입.
  • 이바라키 도지가 먼저 죽으면 분노하며 페이즈 2 돌입.
    • 슈텐 도지의 이속이 빨라지며 스킬들이 강화된다.
    • 스킬 강화 : 발동 시간이 짧아지며 공격의 범위가 넓어진다.
  • 스킬 구성
    • 돌진 : 플레이어 방향으로 빠른 속도로 돌진.
    • 내려찍기 : 커다란 도끼를 내려찍으며 플레이어 위치에 원형 범위 공격을 가함.
    • 화염 방사 : 플레이어 방향으로 원뿔 모양의 화염을 내뿜는다.
    • 불의 손아귀 : 플레이어를 자신의 위치로 끌고 온다.
    • 대폭발 : 이바라키 도지를 자신의 위치로 불러들여 함께 넓은 범위의 대 폭발을 일으킨다.

이바라키 도지

  • 슈텐 도지에게서 일정 범위 이상 벗어나지 않으며 원거리 전투 진행
  • 플레이어가 가까워지면 랜덤 위치로 이동한다.
  • 스킬 구성
    • 원거리 투사체 : 느린 속도로 플레이어를 향해 날아가는 투사체를 발사함.
    • 밀어내기 : 플레이어가 가까워지면 바람을 날려 플레이어를 밀어낸다.
    • 늪지대 생성 : 플레이어 위치에 데미지를 주는 원형 장판을 생성한다.
    • 광역 힐 : 맵 랜덤 위치에 10개의 구체를 생성한다.
    • 제한 시간 내에 구체를 파괴하지 못하면 남아있는 구체 개수에 비례하여 맵 내 몬스터들의 체력을 회복한다.

이벤트 기조

  • 이벤트 기간 동안에만 획득 가능한 한정 아이템 제공.

  • 타깃 유저 : PU 대상 플레이어들에게 더 높은 도달 목표 제공.
  • 이벤트 진행 몇 개월 전에 출시된 신규 서버에는 플레이어들의 성장 속도를 조절하고자 전투 소환수가 미반영된 상태.

    • 전투 소환수 : 무기 (표창)을 장착하여 함께 전투에 참여.
  • 기존에 이벤트에서 다수의 신등급 소환수가 출시되었었고, 신규 소환수로 제공 가능한 버프/디버프 효과가 제한되어 더 이상 플레이어들에게 어필이 되지 않음.
  • 게임 최초로 초월 등급의 소환수 출시를 통해 기존 유저들이 획득하고자할만한 매력적인 아이템을 제공.
  • 초월 등급 소환수는 적에게 대미지와 디버프를 적용하는 스킬을 사용함으로 신규 서버에서 전투 소환수의 부재를 메꿔준다.
  • 초월 등급 소환수의 차별화를 위해 소환수의 스킬 강화 기능 및 소환 시 적용되는 추가 스탯을 적용.

청룡 성장 및 승급

  • 각 등급의 청룡을 레벨 10으로 성장.
    • 소환 상태로 몬스터를 처치하여 소환수 경험치 획득.
    • 청룡 소환수 전용 경험치 아이템을 사용하여 경험치 획득.
  • 청룡 레벨 10 도달 시 아이템을 사용하여 높은 등급으로 승급.
    • 임무 진행 및 상품 구매를 통해 획득한 “여의주” 아이템과 소환수와 동일한 등급의 “소환수 영혼석” 이라는 재료를 소모하여 승급 가능.
    • 예시 : 희귀 등급의 “청룡” 소환수의 레벨 10달성 이후 “여의주” {N}개와 “희귀 소환수 영혼석” {N}개를 소모하여 영웅 등급으로 승급.

초월 등급의 차별점.

  • 초월 등급 달성 시 해금되는 추가 스탯.
    • 청룡 소환 시 적용되는 3개의 스탯이 랜덤으로 결정됨.
    • 각 스탯은 인 게임 아이템을 소모하여 다른 랜덤 스탯으로 변경 가능.
  • 초월 등급 소환수 스킬 강화.
    • 청룡의 기본 스킬은 초월 등급 달성 이후 3개의 스킬 중 하나로 업그레이드 가능.
    • 각 스킬은 인 게임 아이템을 소모하여 해금이 가능하며 한번 해금된 스킬들은 자유롭게 변경할 수 있음.

서바이벌 게임 플레이 방식

  • 하나의 챕터별로 제한 시간 동안 몰려오는 적들을 처치하며 살아남아야 한다.
  • 몬스터를 처치 하여 인 게임 레벨을 올릴 수 있고 증가한 레벨에 따라 사용 가능한 스킬을 획득하거나 강화할 수 있다.
  • 마지막 보스 처치 시 해당 챕터가 클리어 되며 보상을 획득한다.

최초 기획

기획 방향 : 멀티플레이의 재미를 접목시킨 서바이벌 게임 제작.

  • 플레이어들이 서로에게서 멀어지지 않고 협력이 가능하도록 무한 맵이 아닌 제한된 크기의 맵으로 제작.
  • 플레이 흐름이 끊기지 않기 위해 웨이브 형식으로 제작.
    • 레벨 업 시점이 플레이어마다 동일하지 않아, 스킬 강화를 위해 퍼즈를 하지 않는 방향으로 기획.

레벨 업 : 몬스터 처치를 통해 드롭된 경험치를 획득하여 레벨 업.

  • 랜덤으로 스킬을 획득 혹은 보유한 스킬 레벨이 증가한다.
  • 하나의 스킬은 최대 5레벨까지 증가 가능.
  • 25레벨 이후로는 장착한 장비 등의 효과로 인해 스탯 증가 효과만 적용.

웨이브 : 챕터는 5개의 웨이브로 구성.

  • 웨이브가 진행될수록 플레이 타임이 상이하며 제한 시간 동안 몬스터들에게서 살아남아야 함.
  • 웨이브가 넘어가는 구간에서 10초의 “특성 선택” 시간이 주어짐.
  • 5번째 웨이브에서는 보스를 제한 시간 내에 처치하여 챕터 클리어.

스킬 각성 : 각 스킬은 각성을 하기 위해 3개의 특성을 요구.

  • 특성 선택 시간 동안 주어진 3개의 옵션에서 하나를 선택.
  • 3개의 특성을 모두 배우면 스킬은 자동으로 강화 스킬로 변경.
  • 동일 특성을 공유하는 스킬들이 있고, 장비 등의 효과로 인해 필요로 하는 특성을 보다 쉽게 획득 가능.

개선 방향

의도

  • 밸런스 이슈로 인한 콘텐츠 소모 속도 완화 및 Hack 'n' Slash 적인 재미 강화.
    • 권장되는 스펙보다 낮음에도 챕터를 클리어할 수 있는 현상.
    • 특정 스킬만을 배우는 데에 치중되어 획일화된 게임 플레이.

몬스터 소환 방식

  • 기존 : 맵 내 랜덤 위치에서 몬스터 소환.
  • 변경 : 기존 방식과 신규 방식을 챕터 콘셉트 별로 분류하여 사용.
  • 신규 방식 : 해당 챕터는 기존 맵보다 더 넓게 제작되며 몬스터들은 플레이어 위치를 기준으로 원형으로 넓게 소환되어 플레이어에게 조여들어감.
  • 이는 맵 밖에서도 몬스터들이 생성되어 플레이어에게 보다 적극적인 움직임을 격려함.

웨이브 진행 방식

  • 기존 : 제한 시간 동안 살아남아 다음 웨이브로 넘어감.
  • 변경 : 웨이브를 클리어하기 위해 정해진 마릿수의 몬스터 처치.
  • 제한 시간이 지나면 몬스터들이 폭주하며 이동속도가 대폭 증가한다.
  • 폭주 상태는 제한 시간이 없으며 죽지 않고 몬스터 처치 수를 채우면 다음 웨이브로 넘어갈 수 있음.
  • 몬스터들을 처치 못하는 스펙을 가진 유저들은 클리어하지 못하도록 예방하되 다소 애매한 스펙의 유저들은 컨트롤을 통해 통과할 수 있도록 함.

비기 시스템 추가

  • 캐릭터 별로 고유의 특수 스킬을 사용할 수 있는 신규 시스템.
  • 몬스터를 처치하여 비기를 충전할 수 있고 플레이어가 원하는 타이밍에 버튼을 눌러 비기 사용 가능.
  • "비기"를 통해 한 번에 많은 수의 몬스터를 처치할 수 있어 Hack 'n' Slash의 경험을 강화.
