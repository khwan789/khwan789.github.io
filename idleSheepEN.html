Shepherd Girl:
Idle Sheep Farm

‘Shepherd Girl’ is an idle SNG mobile game released in December 2025.


By combining idle gameplay, decoration, and uesr to user visiting environment the game aims to create a community based retention system that differentiates itself from the cozy genre.

Core Loop

  • Goal : “An idle structure that secures session time with low fatigue.”
  • Side content play → Earn currency.
  • Grow through building and animal upgrades.
  • Stay online to earn resources like wool → Use earned currency to decorate the farm.

Economy

Update Direction

Content updates around “stage-based unlocks + repeatable/daily content expansion” to stabilize overall session time and retention.




Core idle mechanics and farm expansion features within the core loop have been completed, but reward content is currently insufficient.


Stage-based content unlocks and content additions.


  • Repeatable tap-based events.
    • Short and instantly rewarding content that can be played frequently.
    • Strengthen monetization through additional rewards through ad viewing.

  • Daily event available once per day.
    • Increasing rewards based on cumulative login days to drive retention.

  • Common : Secure session time across all users.

KPI Goals

Retention

  • D1 30% : Achieve 50% reach rate to the 3rd building
    (drive early gameplay through stage-based unlocks).
  • D7 10% : Return motivation with daily content until player reaches the 5th building.
  • D30 3% : Secure a stable core user base through repeatable daily content.

Session

  • Maintain 15 minutes daily session time.

Monetization

  • Strenghten reason to watch ads and derive ad removal subscription sales.
