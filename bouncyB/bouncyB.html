Bouncy Blast

Bouncy Blast is a mixture of a puzzle game and a brick breaker game.
1000~5000 Downloads.

Bitmango - Puzzle Game Company Intern
June, 2017 ~ August, 2017

The game was created to test how well two different kind of games can work together. As a game designer I worked on designing and balancing the game.

link to: Bouncy Blast - Google Play

Objective

Try to break as many bricks as possible before it reaches the ground. Tap two or more connected blocks in order to fire.
