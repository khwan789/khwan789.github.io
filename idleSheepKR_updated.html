프로젝트 성과 요약

시스템·경제·밸런스·콘텐츠·라이브 지표 분석을 포함한 기획 전반을 담당했습니다.

기간 : 2025.12 ~ 진행 중


주요 지표

  • 다운로드 수 : 500+
  • 매출 : ~200$/month
  • 활성 사용자당 참여 세션 수: 4
  • 세션당 평균 참여 시간: 14분
  • 활성 사용자당 평균 참여 시간: 1시간

리텐션

  • 최고 코호트 : D1 66% / D7 33%
  • 평균 : D1 15% / D7 3%
  • D3 : 7.2%

분석

요약: 초기 몰입(세션/빈도)은 강점이지만, 중기 목표(언락) 부족으로 D7/D30까지 이어지지 않는 구조입니다.

  • 높은 세션 타임과 빈도는 초기 참여도가 높음을 보여줌.
  • 다만 인게임 내 도달 목표가 빠르게 고갈되어 추후 참여도가 빠르게 떨어짐.

양 세는 소녀:
방치형 농장 키우기

‘양 세는 소녀’는 2025년 12월에 출시한 방치형 SNG 장르의 모바일 게임으로,

방치형 + 꾸미기 + 유저 간 방문 구조를 결합해 코지 장르에서 차별화된 커뮤니티 기반 리텐션을 만드는 게임을 목표로 하고 있습니다.




리텐션 개선

라이브 서비스를 통한 데이터 분석 및 회고

코어 루프

  • 의도 : “낮은 피로도로 플레이 타임을 확보할 수 있는 방치형 구조”
  • 사이드 컨텐츠 플레이 >> 재화 획득
  • 건물 업그레이드 및 동물 수집 및 성장
  • 온라인 상태 유지 시 양털을 비롯한 재화 획득 >> 획득한 재화를 통해 농장 꾸미기

진행 구조

초기 목표

  • 플레이어의 50% 이상이 Day 1에 3번째 건물 (외양간) 이상 도달하도록 설계.
  • 안착된 유저 7일 내에 5번째 건물 (놀이터) 도달.

문제 정의

  • 현상 : Day 3 이후 리텐션이 급격히 하락
  • 원인(핵심) : 건물 105 전후부터 ‘새로운 언락/의미 있는 목표’가 줄어들며 목표 지평선이 짧아짐
  • 데이터 해석 : 진행 퍼널은 절벽 없이 완만한 지수형 감소(밸런스 구조는 안정적)지만, 중기 레이어가 얕아 D7/D30로 이어지기 어려움
  • 정리 : 세션 시간/빈도 자체는 강점이며, 개선 포인트는 ‘구조적 변화(언락) 공급’과 ‘중기 목표 설계’

개선 방향

목표 : 의미 있는 성장 목표 구간을 약 3일 → 10~14일로 확장




A. 건물 단계별 시스템 언락 재구성

  • 전환 방향 : “건물 = 숫자 상승” → “건물 = 시스템 언락”
  • 의도 : 성장 과정에서 ‘새로운 경험’을 단계적으로 공급해 목표 지평선을 늘림
  • 언락 예시
    • 103 : 구조화된 일일 액티브 콘텐츠(짧고 명확한 목표형)
    • 105 : 수집/세트 보너스 등 ‘중기 목표’ 레이어
    • 107 : 메타 성장 트리(영구 성장)로 장기 목표 고정
    • 109 : 고급/확장 콘텐츠(이벤트·상위 모드 등)로 10~14일 런웨이 확보



B. 경제 구조 지속가능성 개선

  • 기존 구조 : 온라인 체류 시간이 길수록(켜두기) 재화 효율이 높아지는 구조
  • 리스크
    • 수동 화면 켜두기(패시브 파밍) 유도
    • 피로 누적/번아웃 가능성
    • 구조적 리텐션보다 ‘체류 시간’에 의존하는 지표 왜곡
  • 개선 방향
    • 오프라인 보상 효율 강화 (로그아웃 후 복귀 동기 확보)
    • 사이드 콘텐츠 보상 밀도 강화 (의미 있는 플레이가 가장 효율적)
    • 장시간 온라인 체류는 점진적 효율 감소(완만한 체감)로 조정
  • 목표 : “오래 켜두기”가 아니라 “할 게 있어서 들어오는” 구조로 전환

목표 KPI

리텐션

  • D3 : 초기 ‘구조적 언락’ 밀도 개선 D7 : 메타 성장(영구 성장) 도입으로 중기 목표 고정 D14 : 단계적 레이어 확장으로 장기 계단식 성장

세션

  • 세션당 평균 14분 수준 유지 활성 유저당 일일 4회+ 접속 빈도 유지 ‘장시간 켜두기’ 의존도를 낮추고 의미 있는 플레이로 전환

수익화

  • 마음(동물 성장) 기반 가속 루프 강화 동물 성장의 체감 가치(보상/효율) 상승 양털(꾸미기)은 파워와 분리해 공정성 유지

마무리

  • Sheep Girl은 짧은 성장 구조를 가진 방치형 게임에서 “목표 지평선 확장”을 중심으로
  • 영구 성장 기반으로 중기 retention을 확장하는 방향으로 발전 중입니다.
